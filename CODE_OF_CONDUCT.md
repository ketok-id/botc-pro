# Code of Conduct

This project follows the **Contributor Covenant, version 2.1**. The full text
is published at:

https://www.contributor-covenant.org/version/2/1/code_of_conduct/

By participating in this project (issues, pull requests, discussions, or any
community space associated with it) you agree to abide by that Code of Conduct.

## Scope

It applies to all project spaces, and also when an individual is officially
representing the project in public — including issue comments, pull-request
reviews, and any linked chat or forum.

## Reporting

If you experience or witness behaviour that violates the Code of Conduct,
please report it privately to the project maintainer listed in
[SECURITY.md](./SECURITY.md). Reports will be handled confidentially.

We will not tolerate retaliation against anyone who makes a good-faith report.
