Play Blood on the Clocktower

Unofficial desktop client for LAN and hosted play. Trouble Brewing script included; 5–15 players.

Connect to any ws://host:port or wss://… address. You can deploy the same server (node src/main/server-standalone.js) on a VPS or use a friend's LAN host via port-forwarding or a tunnel.

Quick reference

  • Good wins if the Demon is dead and no Minion can become the Demon.
  • Evil wins when only 2 players remain alive and one is the Demon.
  • The Storyteller runs the night phase, delivers info, and resolves deaths.
  • Dead players keep a single ghost vote for the rest of the game.
  • 5–15 players supported on Trouble Brewing.
